# Salmonator by NPR

Demo comercial de trazabilidad operacional para flujo de salmón:

Guía por correo -> OCR -> lote automático -> recepción Caja OK + pesaje inicial -> cámara/tránsito/descongelado -> semiterminado -> rendimiento -> empaque manual -> despacho con pistoleo -> rebaja de stock.
